# DanielW Skills

A collection of custom skills for AI agents, organized by learning-focused, content-focused, and software-engineering-focused workflows.

## Skill Groups

- **learn-skills**: Skills for research, learning, and content-processing workflows.
- **content-skills**: Skills for content production, media processing, and publishing workflows.
- **swe-skills**: Skills for software engineering workflows, debugging support, and development process helpers.

## Skills

- **learn-skills/youtube-podcast**: Tools and instructions for processing YouTube podcast transcripts and generating mindmaps.
- **content-skills/doubao-asr**: Standalone Doubao ASR workflow for turning local audio or video files into SRT subtitles.
- **swe-skills/check-postmortem**: Review prior bug-fix postmortems under `.agents/postmortem` and surface similar historical fixes before starting a new bug fix.
